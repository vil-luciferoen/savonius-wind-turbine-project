Savonius Wind Turbine Project

Experimental Investigation of 3D Printed Savonius Wind Turbine

Project Overview

This project investigates the design and fabrication of a Savonius wind turbine using additive manufacturing techniques. Multiple blade configurations were developed to study the influence of blade twist angle on turbine performance. The blades were modeled using CAD software and fabricated using PLA material through FDM 3D printing technology.
